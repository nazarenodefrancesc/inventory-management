Request Data Flow — filtering orders by warehouse & month

1
User changes a filterfrontend

A selection in FilterBar.vue updates a shared ref in the useFilters() composable (e.g. selectedLocation = "Tokyo").

2
View reacts & calls the API clientfrontend

The active view reads getCurrentFilters() and calls api.getOrders(filters).

3
HTTP request is issuedfrontend

api.js builds query params and sends GET http://localhost:8001/api/orders?warehouse=Tokyo&month=2025-01.

4
FastAPI endpoint handles itbackend

get_orders() in main.py receives the query parameters.

5
Data is loaded & filtereddata

Orders are read from mock_data.py, then narrowed by apply_filters() and filter_by_month().

6
Response is validated & returnedbackend

The result is serialized as List[Order] via Pydantic and returned as JSON.

7
View stores & rendersfrontend

axios resolves the JSON into a ref; computed properties derive totals; the template renders the filtered rows.
